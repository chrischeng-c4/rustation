# Rush

A modern shell implementation written in Rust, designed to replace traditional shells like zsh, bash, and fish.

## Development

This project is part of the `rust-station` workspace. To build and run:

```bash
# From the workspace root
cargo run -p rush

# Or from this directory
cargo run
```

## Building

```bash
# Debug build
cargo build -p rush

# Release build
cargo build -p rush --release
```

## Testing

```bash
cargo test -p rush
```

## Status

This project is in early development.
